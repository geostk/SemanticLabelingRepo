Contents of Dataset folder:

-> Files_dbpedia_complete: Mixture of textual and numeric semantic labels extracted from DBPedia on City domain
-> Files_dbpedia_numeric: Purely numeric semantic labels extracted from DBPedia on City domain
-> Files_museum: Purely textual semantic labels from Museum domain
-> Files_weather: Data from Weather domain
-> Files_flightstatus: Data from Flight status domain
-> Files_phonedirectory: Data from Phone directory domain
